### 【490-Week 01】第一周学习总结

#### 学习总结

本周主要学习了几个线性数据结构：数组、链表、跳表、堆、栈

数组和链表的对比：

|            | 数组 | 链表 |
| ---------- | ---- | ---- |
| 插入、删除 | O(n) | O(1) |
| 随机访问   | O(1) | O(n) |

本周整体完成的作业比预期的少，只完成了8道题，而且大部分题目都只完成了一遍，没有隔天后再次练习，按照这样的练习强度，学习的效果不会太好。

下周的目标是一周完成20道题以上，并且题目都复习一遍。
