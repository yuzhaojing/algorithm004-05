#使用二分查找，寻找一个半有序数组[4, 5, 6, 7, 0, 1, 2]中间无序的地方
思路如下
1、半有序也即部分有序，仍然可以用二分查找
2、通过判断中间值与头尾数字的大小，得出当前所在的区域，继续套用二分查找模板方法即可
